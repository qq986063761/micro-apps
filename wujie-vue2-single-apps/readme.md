# 跨应用调用组件
  - 子应用利用 webpack5 联邦暴露组件给主应用，就能不加载子应用直接使用组件，缺点是子应用组件需要在组件被使用时自己加载自己应用的公用数据
  - 子应用调用父应用通过 props 注入（可以避免跨域问题）的 `window.$wujie.props.$app` 的 use 方法调用组件
